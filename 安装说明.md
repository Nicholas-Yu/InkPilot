# InkPilot（墨航）安装说明

## 安装步骤

1. 打开终端或文件管理器
2. 将本文件夹复制到你的 Obsidian 插件目录：
   - 目录路径：`你的Vault路径/.obsidian/plugins/`
   - 复制后的目录名应该是 `inkpilot`

3. 打开 Obsidian，进入设置 -> 第三方插件
4. 安全模式关闭后，在已安装插件中找到 "InkPilot"
5. 点击启用按钮

## 插件目录结构确认


## 从源码构建

```bash
# 克隆仓库
git clone https://github.com/nicholasyu/inkpilot.git
cd inkpilot

# 安装依赖
npm install

# 构建
npm run build

# 复制到插件目录
cp main.js manifest.json styles.css 你的Vault路径/.obsidian/plugins/inkpilot/
```

## 配置

启用插件后，进入 Obsidian 设置 -> InkPilot：

1. 选择 API 提供商（OpenAI / Anthropic / 自定义）
2. 填入 API Key
3. 配置模型和参数
